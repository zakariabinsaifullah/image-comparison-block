Beaf is a Gutenberg Block developed with Gutenberg Native Components to showcase photo Comparison with different styles.

== Description ==
**Beaf** is a custom Gutenberg Block based plugin that is developed with **Gutenberg Native Components**. You can easily showcase the **Photo Comparison**. It comes with  different styles and settings so that you can easily customize it as per your needs. It is a lite version of Beaf Pro. To get more features and options, you can check the [Beaf Pro](https://beaf.gutenbergkits.com).

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/image-compare-block` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Simply go to Gutenberg Block Editor and you will find a new block at Media Category named "Image Compare"
4. Drag and drop to use it.


== Frequently Asked Questions ==
= What is Beaf Slider? =
Beaf slider means Before and After Slider. It is a slider that showcases the before and after comparison between two images, videos, or iframes.

= What is Beaf Pro? =
Beaf Pro is a premium version of Beaf. You can get more features and options in Beaf Pro. [Get Beaf Pro](https://beaf.gutenbergkits.com)

= How to upgrade to Beaf Pro? =
Beaf Pro is the premium version of Beaf and it comes with a complete set of features and options. It is completely a separate plugin. You can simply deactivate the Beaf and install Beaf Pro. Then, you can start using it. 

= Is it compatible with the latest WordPress version? =
Yes, it is compatible with the latest WordPress version.

= Is it responsive? =
Sure, it is 100% Responsive

= Have you used any block builder? =
No, it is built with 100% Gutenberg Native Components only.
